# Vectors Raycasting (Manim)

Small Manim scenes demonstrating 2D vectors, rotation, and raycasting concepts.

## Requirements

- Python 3.9+
- pip

## Install

1. Create and activate a virtual environment (recommended).
2. Install dependencies:

```
pip install -r requirements.txt
```

## Run

Render the main scene from `final.py`:

```
manim -pqh final.py All
```

- `-p` opens the output when done.
- `-q h` renders at high quality (use `-q l` for fast previews).

## Files

- `final.py`: main animation.
- `tuto.py`: scratch/testing scene.

## Notes

- If Manim is missing system dependencies on Linux, follow the official install guide: https://docs.manim.community/en/stable/installation.html
